# lp-kids-teeth-20260326

Vercel に公開されていた `pediatric-orthodontics.86kenkyujo.com` の配信 HTML から復元した静的サイトです。

## 構成

- `index.html`: 公開サイトから取得した単一 HTML
- `vercel.json`: Vercel の静的配信用設定

## 注意

これは GitHub の元リポジトリから clone したものではなく、公開 URL で配信されている HTML をもとに復元したプロジェクトです。
元の Vercel プロジェクトソースを取得するには、`kyoshirotaniguchis-projects` にアクセスできる Vercel アカウントで CLI ログインが必要です。
